Simple Pos System

Github: https://github.com/danielyap92/simple-pos-system

This is a very simple POS system webpage create using React.

In this webpage, user can select item to add to cart, it will calculate the total amount and allow user to choose payment method. Once the order is finalized, click on the “pay now” will generate a receipt with unique ID and time stamp.

Build by: REACT, bootstrap

Simple Recipe

Github: https://github.com/danielyap92/recipe

A simple recipe website powered by API.

This webpage contain of meal recipe info from API , allowed user to search by keyword and category, or show a random recipe.

Build by: Node Express, API

IMS – Inventory Management System

Github: https://github.com/danielyap92/ims

This is a very simple inventory management system web application.

It allow user to register item details and perform stock transaction. All transaction data are stored in database and record can be retrieve as stock card.

Build by: Node Express, postgresSQL

My Car Service Web Application

Github: https://github.com/danielyap92/CS50-Final-project

This is my final project for my first computer science program – Harvard CS50.

This is a web application which can track and monitor car service record. It will suggest next service item based on services interval and allow user to perform create, read and delete service record.

Build by: Python Flask, sqlite3
