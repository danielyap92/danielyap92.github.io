My Journey to Coding

I first learn the computer science since June 2022.
After research, I had chosen to start my programming journey with Harvard CS50x Introduction to Computer Science offered by edX. However I only managed to complete the first 4 chapter (Week 0 to Week 3), and wanted to give up, as the problem set is too difficult, without adequate exposure in programming, it was very difficult for a beginner like me to solve the problem set by writing the code myself. Just imagine you was taught to write “hello world”, and your problem set expect you to build a pyramid with different floor level based on your input number. I had stop learning a while since Dec 2022.
Later in Sept 2023, I resume back this program. I changed my mindset, I admit the fact that I was not able to solve this problem set on my own, I stay humble to look for solution. I study every step in the solution and try to understand the logic behind it. Things are not perfect, but it works. I managed to complete CS50x and create my own final project at Dec 2023.
After completed CS50x, I continued my journey in web development by taking Udemy course The Complete 2023 Web Development Bootcamp by Dr. Angela Yu.
This course is much more easier and beginner friendly compared to CS50x. Basically it covered all the basic in web development (both frontend and backend). I finished this course at Feb 2024, then I start to write own project and continue to explore other technique which are not covered in this course.
